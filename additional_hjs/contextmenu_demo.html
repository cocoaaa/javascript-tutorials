Another example: contextmenu

Contextmenu allows to add customized menu list (for example, when you click on the webpage).

Note: Only FireFox supports this feature for now. Please use Firefox to see how this demo works!

Right-click on this container to see a menu to go to other websites.

You can choose the color of this line by right-clicking -> color from the menu.

Double-click on the image to make it disapper/reappear!
